== Description ==
Adds After The Deadline proofreading for BuddyPress Groups, Forums, Activity Stream, Profiles, Messages and Blog Comments, by adding a nice "spellcheck" button.

Originally built for the <a href="http://wpnyc.org/">WordPressNYC BuddyPress site.</a>

== Installation ==

1. Simply add this plugin normally through the WordPress "Add Plugins" page and activate.
2. There are no options to configure.

== Frequently Asked Questions ==

= What does this plugin do? =
Adds After The Deadline proofreading for BuddyPress Groups, Forums, Activity Stream, Profiles, Messages and Blog Comments.  Users just click on the Spellcheck button to start the proof reader.

= Do I need After The Deadline installed on my site? =
No. This plugin includes all the After The Deadline code you will need.

= Will this conflict with the After The Deadline plugin? =
No. You can still use After The Deadline to write your blog posts.

= Can I see a live demo? =
Absolutely! This plugin was originally built for the <a href="http://wpnyc.org/">WordPressNYC BuddyPress site</a>, and is installed there.  WPNYC.ORG only allows members to post, so you will have to register if you want to see the plugin in action.
