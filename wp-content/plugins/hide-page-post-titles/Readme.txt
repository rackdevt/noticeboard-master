== Description ==
Allows authors to hide the page & post title.

== Installation ==
Following steps:
1. Upload
2. Activate
3. Go to the Page or Post and click on what you want to edit or something new.
4. Below you will find Hide title and click it.
5. Save Page or Post and review the changes.
