== Description ==

This Plugin Will enable short code in WordPress sidebar Widget. By default, wordpress doesn't support Short Code in Sidebar Widget. This Plugin can remove that restriction.
Just Use a text widget for sidebar and use [short code] and ENJOY.

If you like this, Please rate it.


== Installation ==



1. Upload `shortcode-sidebar.php` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
